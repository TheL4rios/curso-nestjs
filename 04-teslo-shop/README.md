# Teslo API

1. Clonar proyecto
2. ```npm i```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variales de entorno
5. Levantar la BD
```
docker-compose up -d
```

6. Levantar: ```npm run start:dev```
